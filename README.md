Scripts to calculate structure-function coupling and to run GWAS.

The folder SFC_calc contains scripts for group comparison analysis. These scripts can be used on the full cohort with imaging data to calculate structure-function coupling as a phenotype for GWAS.

The folder GWAS_scripts contains scripts to run QC and GWAS. array_qc creates snp list for step 1 and impute_qc creates list for step 2.
